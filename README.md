# EXAM HTML5 Game - Polarity Twins

## Ejecutar
Abrir con Live Server o cualquier servidor estático.
Entrar a index.html.

## Controles
- Flechas: mover personaje 1
- WASD: mover personaje 2
- Espacio: campo magnético
- P: pausa
- M: mute

## Estructura
assets/
engine/
game/puzzle/
net/
index.html
main.js
